Projeto desenvolvido em Java utilizando o padrão de projeto Facade aplicado em um sistema de hamburgueria.

O objetivo é demonstrar como simplificar o acesso a um sistema complexo através de uma fachada centralizadora.
O padrão Facade fornece uma interface simplificada para um conjunto de subsistemas.

Ao invés do cliente acessar diretamente:

cozinha
pagamento
entrega

ele utiliza apenas a fachada principal do sistema.
